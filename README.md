1. Photoalbum app where photos are displayed in masonry block.
2. API call from pixabay api.
3. Built using react and bootstrap.
4. The searched keywords for photos are stored in local storage, upto max of 5 keywords are stored.
5. The number of views is displayed on the photo through the api.
6. If the user clicks on any photo, we show the expanded view in a popup, the user can download the photo.
